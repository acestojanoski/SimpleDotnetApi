# Simple .NET Core API

Simple .NET Core API, used for experiments.

## Start

Using Powershell:
```
git clone https://github.com/acestojanoski/SimpleDotnetApi.git; `
    cd SimpleDotnetApi; `
    ./run.ps1
```

Using Bash:
```
git clone https://github.com/acestojanoski/SimpleDotnetApi.git && \
    cd SimpleDotnetApi && \
    ./run.sh
```

OR:

```
git clone https://github.com/acestojanoski/SimpleDotnetApi.git && \
    cd SimpleDotnetApi && \
    dotnet restore && \
    dotnet build && \
    dotnet run -p SimpleDotnetApi
```
